# node-base-installer

For unix-based OS:

**clone the repository:**

```
git clone https://github.com/santidesimone/node-base-installer.git
```

**to install globally (and locally) this npm module, run:**

```
cd node-base-installer
```

```
sudo npm i -g $(pwd)
```
**to uninstall:**

sudo npm uninstall -g $(pwd)
